📚 Complete HTML Tags Reference

Your complete guide to all HTML tags with descriptions and examples

🎯 "<H1> This tag helps to enlarge any text." - Like this format!

<h1> to <h6>
🔤 Description: These tags help to create headings. <h1> is the largest and most important, <h6> is the smallest.
<h1>This is main heading</h1>
<h2>This is subheading</h2>
<h3>This is smaller heading</h3>
📺 OUTPUT:

This is main heading

This is subheading

This is smaller heading

<p>
🔤 Description: This tag helps to create paragraphs of text. It automatically adds space before and after.
<p>This is a paragraph of text. It can contain multiple sentences and will automatically wrap to the next line.</p>
📺 OUTPUT:

This is a paragraph of text. It can contain multiple sentences and will automatically wrap to the next line.

<b> or <strong>
🔤 Description: These tags help to make text bold. <strong> also adds importance for SEO.
<b>This text is bold</b>
<strong>This text is important and bold</strong>
📺 OUTPUT: This text is bold
This text is important and bold
<i> or <em>
🔤 Description: These tags help to make text italic. <em> adds emphasis meaning.
<i>This text is italic</i>
<em>This text is emphasized</em>
📺 OUTPUT: This text is italic
This text is emphasized
<u>
🔤 Description: This tag helps to underline text.
<u>This text is underlined</u>
📺 OUTPUT: This text is underlined
<mark>
🔤 Description: This tag helps to highlight text like a marker pen.
<mark>This text is highlighted</mark>
📺 OUTPUT: This text is highlighted
<small>
🔤 Description: This tag helps to make text smaller than normal.
<small>This text is smaller</small>
📺 OUTPUT: This text is smaller
<base>
🔤 Description: This tag helps to specify the base URL/target for all relative URLs in a document.
<base href="https://www.example.com/" target="_blank">
📺 OUTPUT:
[Sets base URL for all links - not visible on page]
<code>
🔤 Description: This tag helps to display computer code text in a monospace font.
<code>const x = 10;</code>
📺 OUTPUT: const x = 10;
<pre>
🔤 Description: This tag helps to display preformatted text exactly as written (preserves spaces and line breaks).
<pre>
Line 1
Line 2 (indented)
</pre>
📺 OUTPUT: 
  Line 1
    Line 2 (indented)
<kbd>
🔤 Description: This tag helps to define keyboard input, showing keys users should press.
Press <kbd>Ctrl</kbd> + <kbd>C</kbd>
📺 OUTPUT: Press Ctrl + C
<samp>
🔤 Description: This tag helps to show sample output from a computer program.
<samp>Error: File not found</samp>
📺 OUTPUT: Error: File not found
<var>
🔤 Description: This tag helps to define variable names in mathematical expressions or programming.
<var>x</var> + <var>y</var> = 10
📺 OUTPUT: x + y = 10
<blockquote>
🔤 Description: This tag helps to create a block-level quotation with indentation.
<blockquote>
This is a long quotation.
</blockquote>
📺 OUTPUT:
This is a long quotation.
<q>
🔤 Description: This tag helps to create inline quotations with automatic quote marks.
<q>Short quote</q>
📺 OUTPUT: Short quote
<cite>
🔤 Description: This tag helps to cite the title of creative work (book, song, painting).
<cite>The Mona Lisa</cite> by Leonardo da Vinci
📺 OUTPUT: The Mona Lisa by Leonardo da Vinci
<address>
🔤 Description: This tag helps to display contact information (address, email, phone).
<address>
John Doe
john@email.com
</address>
📺 OUTPUT:
John Doe
john@email.com
<abbr>
🔤 Description: This tag helps to define abbreviations with optional tooltip on hover.
<abbr title="World Health Organization">WHO</abbr>
📺 OUTPUT (hover over WHO): WHO
<bdo>
🔤 Description: This tag helps to change text direction (left-to-right or right-to-left).
<bdo dir="rtl">This text is backwards</bdo>
📺 OUTPUT: This text is backwards
<fieldset> and <legend>
🔤 Description: <fieldset> groups related form elements. <legend> gives it a caption.
<fieldset>
<legend>Personal Info</legend>
Name: <input>
</fieldset>
📺 OUTPUT:
Personal Info Name:
<label>
🔤 Description: This tag helps to define labels for form elements, making them clickable.
<label for="name">Name:</label>
<input id="name">
📺 OUTPUT (click on "Name:"):
<datalist>
🔤 Description: This tag helps to provide autocomplete options for input fields.
<input list="browsers">
<datalist id="browsers">
<option value="Chrome">
</datalist>
📺 OUTPUT (start typing):
<optgroup>
🔤 Description: This tag helps to group options in a dropdown list.
<select>
<optgroup label="Fruits">
<option>Apple</option>
</optgroup>
</select>
📺 OUTPUT:
<output>
🔤 Description: This tag helps to display the result of a calculation.
<output name="result">0</output>
📺 OUTPUT: Calculation result appears here
<progress>
🔤 Description: This tag helps to show progress of a task (like download percentage).
<progress value="70" max="100">70%</progress>
📺 OUTPUT:

70% complete

<meter>
🔤 Description: This tag helps to display a scalar measurement (like disk usage, temperature).
<meter value="0.6">60%</meter>
📺 OUTPUT:

60% used

<thead>, <tbody>, <tfoot>
🔤 Description: These tags help to group table header, body, and footer sections.
<table>
<thead><tr><th>Header</th></tr></thead>
<tbody>...</tbody>
<tfoot>...</tfoot>
</table>
📺 OUTPUT:
Header
Body content
Footer

<th>Name</th>
<th>Age</th>
📺 OUTPUT:
Name Age
<colgroup> and <col>

<colgroup>

</colgroup>
📺 OUTPUT:

<caption>

<figure> and <figcaption>

<figure>

</figure>
📺 OUTPUT:

🔤 Description: This tag helps to provide different images for different screen sizes.
<picture>
<source media="(min-width: 650px)" srcset="big.jpg">
<img src="small.jpg">
</picture>
📺 OUTPUT:
🖼️ [Responsive image based on screen size]
<svg>
🔤 Description: This tag helps to draw scalable vector graphics.
<svg width="100" height="100">
<circle cx="50" cy="50" r="40" fill="blue" />
</svg>
📺 OUTPUT:

📺 OUTPUT:

<iframe src="https://example.com"></iframe>

<embed>
🔤 Description: This tag helps to embed external content like PDFs or Flash (older method).

📺 OUTPUT:
📄 [Embedded PDF would appear here]
<object>
🔤 Description: This tag helps to embed objects like PDFs, videos, or Java applets.

📺 OUTPUT:
📦 [Embedded object would appear here]

🔤 Description: This tag helps to pass parameters to embedded objects.

<param name="autoplay" value="true">
</object>
📺 OUTPUT:
⚙️ [Parameters passed to embedded object]
<track>
🔤 Description: This tag helps to add subtitles or captions to video/audio.
<video>
<track src="subtitles.vtt" kind="subtitles">
</video>
📺 OUTPUT:
📝 [Subtitles would appear with video]
<details> and <summary>
🔤 Description: These tags help to create expandable/collapsible content sections.
<details>
<summary>Click to expand</summary>
<p>Hidden content</p>
</details>
📺 OUTPUT (click to expand):
👉 Click to expand

✨ Hidden content appears here! This is collapsible.

<dialog open>This is a dialog</dialog>
📺 OUTPUT:

This is a modal dialog example
